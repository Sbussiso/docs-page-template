Welcome to SourceBox!

SourceBox is a machine learning platform for both developers and consumers alike. Use our demos to get the most out of LLM's beyond ChatGPT and similar models. Harness the power of frameworks such as langchain and platforms like huggingface for a uniqe LLM experience. Spin up chat bots quick and easy with WikiDoc for custom Q/A from multiple data sources of different types including databases. Automate parts of LLM development with tools like CodeDoc. Use tools like Source-Lightning to download instantly executable UI included models to run and test localy fast as lightning! Do all this and more all in one place with SourceBox!

Pack-Man is a service for creating, storing, and managing "packs" of custom data to use dynamicaly on various services across the SourceBox platform! For example you can connect packs to Source-Lightning and WikiDoc for easy custom Q/A, downloadable models, and templates with your own data included.
  • Packs are split into 2 catagories "SourcePacks" and "CodePacks"
  • Connect packs to platform services easily

From instantly executable no code applications to robust starter code templates, start and finish your LLM projects in a flash with Source-Ligtning.
  • Save both time and effort using and developing LLM applications
  • Build, download, and run custom LLM's easily and locally with a UI of your choice
  • Effortlessly integrate LLM's into your coding projects with our out of the box downloadable lightning-plates
